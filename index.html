Wolfe

Declarative Machine Learning in Scala

Welcome

Wolfe acknowledges that at the heart of most machine learning (ML) applications are two concepts: mathematical models that assign probabilities or scores to hypotheses (or possible worlds), and mathematical operators that interact with these models---most prominently maximization and summation.

In Wolfe these mathematical operators are primitives, and the practitioner can rely on the Wolfe interpreter to implement these in optimized, and generally approximate, fashion. This enables declarative and concise development of ML applications, which allows the practitioner to focus on mathematical modelling of her problem as opposed to machine learning implementation details. With these primitives, existing probabilistic programming and Statistical Relational Learning approaches (e.g. Factorie or Markov Logic) can be used and extended.

Wolfe supports (or will support) most objectives and operations used in learning and inference: convex learning risks functions, linear programs for MAP inference, integer linear programming solvers, message passing, nested min max objectives, dual decomposition etc. Wolfe’s host language is Scala.
